# [개발 기획서] 링크 포켓 (Link Pocket) - 우클릭 간편 북마크 저장소

## ① 서비스 정의

- **무엇을 만드는가?**
  - 웹 서핑 중 나중에 다시 읽고 싶은 뉴스 기사, 블로그, 유용한 웹사이트 링크를 마우스 우클릭 한 번으로 빠르게 분류하고 저장하는 크롬 확장 프로그램(Chrome Extension)입니다.
- **누구를 위해 만드는가?**
  - 인터넷 자료 조사를 자주 하지만 브라우저 기본 북마크 기능이 복잡하고 무거워 불편함을 느끼는 학생, 직장인, 개발자들을 위해 만듭니다.
- **기획 배경:**
  - 기존 북마크는 폴더 지정이 번거롭고, '나중에 읽어야지' 하고 열어둔 탭이 많아지면 브라우저가 느려집니다. 우클릭 메뉴를 통해 현재 페이지를 즉시 저장하고, 팝업 UI에서 직관적으로 관리할 수 있는 가벼운 포켓 저장소가 필요하여 기획되었습니다.

---

## ② 기능 정의

- **[필수] 우클릭 메뉴 생성 (`contextMenus`)**
  - 웹페이지 빈 곳을 우클릭하면 '링크 포켓에 저장' 메뉴가 나타납니다.
  - 마우스를 올리면 하위 카테고리 메뉴(예: 개발/정보, 뉴스, 쇼핑, 기타)가 표시되어 원하는 분류로 즉시 저장할 수 있습니다.
- **[필수] 데이터 백그라운드 처리 (`background.js`)**
  - 사용자가 우클릭 메뉴를 클릭하면, 백그라운드 스크립트가 실행되어 현재 활성화된 탭의 'URL'과 '웹사이트 타이틀' 정보를 동적으로 수집합니다.
- **[필수] 데이터 저장 (`chrome.storage`)**
  - 백그라운드에서 수집한 링크 데이터를 브라우저 로컬 저장소(`chrome.storage.local`)에 배열(Array) 형태로 영구 저장합니다. 브라우저를 닫아도 데이터가 유지됩니다.
- **[필수] UI 형태 (Popup UI)**
  - 확장 프로그램 아이콘을 누르면 깔끔한 리스트 형태의 팝업 창이 열립니다.
  - 저장된 링크들이 카테고리별로 분류되어 나타나며, 클릭 시 해당 페이지로 바로 이동하고 '삭제(X)' 버튼을 통해 쉽게 지울 수 있습니다.

  \*\*(`content.js`) background.js로부터 "링크 저장 완료" 메시지를 받으면,
  현재 사용자가 보고 있는 웹페이지의 우측 상단에
  동적으로 토스트(Toast) 알림 창을 띄워주는 역할을 한다.

## ③ UX 설계 (사용자 시나리오)

1. **링크 수집 단계:**
   - 사용자가 유용한 블로그 글을 읽다가 마우스 우클릭을 합니다.
   - '링크 포켓에 저장' -> '개발/정보' 카테고리를 차례로 클릭합니다.
   - 화면 우측 상단에 "링크가 포켓에 저장되었습니다!"라는 작은 안내 알림(Notification)이 뜹니다.
2. **링크 확인 및 관리 단계:**
   - 나중에 저장한 링크를 보고 싶을 때, 브라우저 우측 상단의 '링크 포켓' 아이콘을 클릭합니다.
   - 팝업 UI가 열리며 '개발/정보' 탭에 아까 저장한 블로그 글의 제목이 리스트로 보입니다.
   - 리스트를 클릭하면 새로운 탭이 열리며 해당 사이트로 이동합니다. 다 읽은 링크는 우측의 '삭제' 버튼을 눌러 정리합니다.

## ④ 기술 명세

- **기본 개발 환경:** HTML5, CSS3, JavaScript (Vanilla JS)
- **확장 프로그램 아키텍처:**
  - `manifest.json`: 확장 프로그램의 메타데이터 정의 및 필수 권한 선언 (`contextMenus`, `storage`, `tabs`)
  - `background.js`: 우클릭 메뉴 생성 규칙을 작성하고, 메뉴 클릭 시 현재 탭의 `title`과 `url`을 받아와 `chrome.storage`에 넘겨주는 핵심 백그라운드 로직 구현
  - `popup.html` & `popup.js`: 사용자가 아이콘을 눌렀을 때 나타나는 UI 화면으로, 저장된 데이터를 화면에 바인딩하고 삭제 및 이동 이벤트 처리
- **핵심 Web Extension API:**
  - `chrome.contextMenus.create()`: 마우스 우클릭 메뉴 구성
  - `chrome.contextMenus.onClicked.addListener()`: 메뉴 클릭 이벤트 감지
  - `chrome.storage.local.set() / .get()`: 데이터 저장 및 로드
