# 📂 링크포켓 (LinkPocket)

> 넘쳐나는 웹 서핑 속, 나만의 링크를 한곳에 쏙!  
> **링크포켓**은 중요한 웹 링크와 북마크를 쉽고 빠르게 저장, 분류, 관리할 수 있는 링크 관리 서비스입니다.

---

## ✨ 주요 기능 (Key Features)

- **간편한 링크 저장**: URL 입력만으로 웹 페이지의 타이틀과 썸네일을 자동으로 파싱하여 저장합니다.
- **카테고리 및 태그 관리**: 폴더(포켓) 지정 및 태그 기능을 통해 링크를 체계적으로 분류합니다.
- **스마트 검색**: 저장된 링크의 제목, 설명, 태그를 기반으로 원하는 링크를 빠르게 찾아냅니다.
- **기기 간 동기화**: 반응형 웹 디자인을 지원하여 PC, 태블릿, 모바일 어디서나 동일한 환경을 제공합니다.
